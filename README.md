# App template — design_space.studio

Reusable scaffold for Next.js apps: conventions, Cursor rules, folder layout, and design-token workflow.

## Stack

See `.cursor/rules/stack.md` (Next.js App Router, Tailwind, Shadcn/ui, Phosphor, Supabase, TypeScript).

## Before you build UI

1. Read `design.md` and fill in tokens.
2. Implement variables in `src/styles/index.css`.

## After cloning

1. Copy `.env.example` to `.env.local` and set values (never commit `.env.local`).
2. Install dependencies when you add `package.json` / lockfile (`npm install`, etc.).

## Repo layout

See **Folder structure** in `.cursor/rules/stack.md`.
